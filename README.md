#WD_01

## Task 1: Interactive Navigation Menu

This project implements an interactive and fixed navigation menu as part of the Web Development Internship at Prodigy InfoTech.

The navigation bar changes its style when the user scrolls the page and when hovering over menu items. It remains visible at the top of the page at all times and works across all sections.

---

## Features
- Fixed navigation bar
- Color/style change on scroll
- Hover effects on menu items
- Responsive design for different screen sizes

---

## Technologies Used
- HTML
- CSS
- JavaScript

---

## How to Run
1. Download or clone the repository
2. Open `index.html` in any web browser
3. Scroll and hover over the navigation menu to see the effects

---

## Learning Outcome
- Improved understanding of DOM manipulation
- Practical use of CSS hover and scroll effects
- Creating responsive and interactive UI components
